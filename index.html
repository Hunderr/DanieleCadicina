Cosa utilizzo




HTML 5

Lo standard dal web 1.0 in poi, è alla base di qualunque progetto passato, presente e futuro. Estremamente performante in velocità, poiché non deve interagire con basi di dati e tutte le sue informazioni si trovano all'interno della pagina.

CSS 3

Lo standard attuale da accompagnare all'html 5. Il CSS è un linguaggio di programmazione web utilizzato per descrivere l’aspetto e la formattazione di un sito web. Permette di creare pagine web ricche di contenuti con requisiti di codice relativamente leggeri. Ciò comporta che gli effetti visivi siano più elaborati, le pagine più leggere, una migliore interfaccia utente, e caricamento delle pagine più veloce.

Javascript

É il linguaggio di programmazione web più popolare e usato al mondo, permette la creazione di effetti dinamici interattivi innescati in vari modi dall'utente sulla pagina web in uso. Inoltre è supportato dai maggiori browser.
